🔬 产品创新的科学基础

烟卷（花）的本质

什么是"花"？

烟卷 = 烟草 + 香料的复合体

核心特性：
✓ 多香料叠加 → 形成风味金字塔
✓ 缓慢燃烧 → 香味逐步释放
✓ 长时间体验 → 渐进式的感官享受
✓ 质感强 → 仪式感和品质感

Vape液体（油）的本质

什么是"油"？

Vape液体 = 尼古丁溶液 + 风味添加剂

核心特性：
✓ 快速雾化 → 10秒内吸收
✓ 迅速上头 → 即时的快感反馈
✓ 高效递送 → 强烈的满足感
✓ 便捷使用 → 无需点火

⚗️ 烟卷+Vape同吸的化学反应

为什么可以同时吸？原理是什么？

吸食烟卷

从烟卷中吸入：烟草香 + 香料香 + 薄荷香等

↓

同时吸食Vape

从Vape中吸入：尼古丁 + 液体风味

↓

风味叠加 × 快感倍增

烟卷的多维风味 × Vape的快速递送

形成1+1>2的协同效应

同吸的科学逻辑

烟卷提供： 风味层次感（嗅觉+味觉）

+ Vape提供： 快速上头感（神经反馈）

= 超感官体验： 多感官同时刺激
